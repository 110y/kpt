## cat

Print Resource Config from a local directory.

![alt text][tutorial]

    kpt tutorial cfg cat

[tutorial-script]

### Synopsis

    kpt cfg cat DIR

  DIR:
    Path to local directory.

### Examples

    # print Resource config from a directory
    kpt cfg cat my-dir/
