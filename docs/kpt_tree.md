## kpt tree

Display package Resource structure

### Synopsis

Display package Resource structure.

  DIR:
    Path to local package directory.


```
kpt tree DIR [flags]
```

### Examples

```
# print package structure
kpt tree my-package/

```

### Options

```
  -h, --help                  help for tree
      --include-subpackages   also print resources from subpackages. (default true)
```
